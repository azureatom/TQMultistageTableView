TQMultistageTableView
=====================

## Overview

**TQMultistageTableView**是一个分3层的列表控件，第一层点击后，可展开第二层列表，第二层点击可展开第三层视图。层之间展开和关闭是互斥的，同时只有一个展开的列表层级。支持最低IOS版本为 **IOS6.0** 。
